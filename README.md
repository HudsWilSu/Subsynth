## Setup

- Install JUCE
- Clone repo
- Open the `.jucer` file with the Projucer
- Launch editor from inside Projucer 
- Build
- Navigate to `./Builds/YOUR EDITOR/x64/Debug/Standalone Plugin/`
- Launch `Subsynth` executable (or w/e it is on MacOS)
